* Python 3.x
* Root Privilages

## Steps (For Linux):
* Clone the Repository 
```shell
git clone https://github.com/manakpandey/isaa_proj.git
```
* Go in the project directory
```shell
cd isaa_proj
```
* Run the following command
```shell
./setup.sh
```
